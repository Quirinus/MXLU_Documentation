Mystic Orbs

These reagents can be bought from any magic items vendor or by killing monsters. Mystic Orbs add a magical bonus to an item at the cost of a higher required level to use the item.

Any non-ethereal equippable item + Mystic Orb → add orb bonus and +2 required level

You can only apply up to 15 of the same Mystic Orb to your item.
Any Mystic Orb effects applied on a runeword are lost upon unsocketing, but the effects applied to the base item stay.

Warning! The level requirement penalty is applied after taking into account both the base item reqired level and the level of any socket fillers. This means for example if you have a belt with required level 20, add five orbs (level penalty +10) and socket a Rha rune (required level 100), you get a belt with required level 100+10 = 110. For that reason, to avoid going over required level 120 on accident, always add the socket fillers before adding orbs.

Oskills
If an item has a skill bonus with no character class requirement (eg. +3-6 to Summon Shadows), then the bonus is an 'oskill', giving this skill to any class for free.

NOTE: Not all Mystic Orbs are for sale. There are 70 known types in circulation, but rumours persist of lost ancient orbs with miraculous powers...
