SHRINE BONUSES LIST

This is a list of preset modifiers that can spawn on crafted items when shrine crafting or on rare, honorific and crafted items when shrine blessing.

What affixes will the item get when shrine crafting?
The resulting crafted item spawns with 0-3 preset crafting affixes (seen in the table below) depending on luck and shrine type used, and 1-4 extra random rare affixes.

See THIS PAGE for more information about random rare affixes when shrine crafting.

What affixes will the item get when shrine blessing?
The item retains its original affixes and gains 0-3 preset affixes depending on luck and shrine type used. You can see them in the table below.

Shrine crafting and blessing bonuses can stack when using the same shrine on an item, for a possibility for a double stat.

See THIS PAGE for more information about shrine crafting and shrine blessing.

You can:

  • Sort the table by clicking a column name. Shift+click allows you to sort by multiple columns.
  • Highlight a row and get column names by moving the mouse over them.
  • Get table column explanations by moving the mouse over the column names.
