This is a list of Monster availability and their Skills and Resistances. Some of these monsters can be summoned as minions and some can be found as reanimates.

The 'Reanimate?' column indicates reanimates from:

  • Two rare/crafting affixes and the upgraded Harkon Crystal Gem (1336-1553 as Affix/Gem)

  • Atanna's Key SSU Jared's Stone (1336-1566 as A)

Reanimates from other sources are not listed here.

NOTE: These are MedianXL 2012 stats, so they are not up to date. There are different variations of monsters with the same name, but different id, skills, resistances and other, so make sure you get the right one when searching.
